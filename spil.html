Opgavens navn 04.05.01 // Individuelt spil
Værktøjer HTML // CSS // Javascript // Adobe Illustrator // Adobe XD
Individuel/gruppe opgave Individuel

Krav

  • Animationerne skal være CSS keyframe animationer
  • Der skal være lyd, som enten selv er produceret eller rettighedsfrit
  • Der skal være en game over- og level complete-lyd
  • Spillet skal bygges op af HTML, CSS, Javascript og SVG (png/jpg til baggrund hvis nødvendigt)

Opgavens fokus

Opgaven gik ud på at animere spil-elementerne i en kombination af CSS og Javascript. Ydermere skulle spillet indeholde musik, der var rettighedsfrit og lydeffekter, der selv var optaget.

Overblik

Jeg lavede et state machine diagram for yderligere at planlægge spillets flow. State machine diagram minder om et aktivitetsdiagram, men er meget mere detaljeret. Her samles flere aktiviteter i hver deres ”state”, som også betegnes som en tilstand. Et spil er i en tilstand, når det udfører de samme ting gentagende gange, indtil der sker noget nyt. Animations- og funktionsnavnene bliver ligeledes fastlagt og det bliver derfra tydeliggjort, hvordan flowet skal forløbe. Jeg fik kendskab til basale animations-principper i kombinationen af CSS og Javascript, som jeg brugte til at animere mine grafik-elementer og skabe et interaktivt spil. Animationerne vi har lært på dette tema, er ikke kun vigtigt at have viden om for at kunne lave et spil. Dette kan også bruges til at fremstille interaktive brugergrænseflader, der engagerer og motiverer brugerne og ligeledes bringer liv til et website.
